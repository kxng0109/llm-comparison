# LLM Comparison App - Frontend

A modern React application for comparing responses from up to three different LLMs.

## Tech Stack
- React 18
- Vite
- TailwindCSS
- Axios (for API calls)

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Configuration

Update the API endpoint in `src/services/api.js` to match your Spring Boot backend URL.

Default: `http://localhost:8080/api`
```
